# Super Easy Html Template Engine

### WHY WE NEED ? ###
---
While we're designing HTML/CSS pages as HTML designer, we need to repeat some HTML blocks which is very tedious and frustrating job. To avoid repeating our html blocks on different pages, I created this **Super Easy HTML Template Engine**.

### FEATURES ###
---
- Easy to reuse HTML blocks/sections.
- Don't need to copy-paste your same HTML blocks to present list or grid elements.
